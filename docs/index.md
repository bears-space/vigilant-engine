# Vigilant Engine

Vigilant Engine is the ESP-IDF framework for Aerobear ESP-based nodes. It provides a stable base for
networked devices with OTA updates, multiple Wi-Fi modes, a web configuration surface, and a shared
peripheral layer.

## What this documentation covers

- How to build and flash main and recovery firmware
- The repo layout and where to add new features
- Runtime capabilities such as OTA and network modes
- Peripheral support and integration patterns
